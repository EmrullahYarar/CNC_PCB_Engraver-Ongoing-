# CNC_PCB_Engraver
3 axis CNC engraver for double sided PCBs.

The project fully 3D printed parts. 8mm mils and NEMA 17 steppers are used for the axises. The project base is GRBLHal. The spindle motor is not attached X or Y axis to minimize the ossilation of spindle motor. The spinle is fixed to working bench with Y axis linear guide. The X&Y axis are stack together. The machine looks like 90 degree rotated 3D CNC.
